# BooAI Mini App

BooAI is a Farcaster mini app that lets creators boost posts with BOOAI tokens and rewards engaged users equally after 24-hour verification.

Run locally:

```bash
npm install
npm run dev
```

This repository includes:
- Next.js (App Router) + TypeScript
- Tailwind CSS
- API routes for manifest and webhook
- Payment & escrow logic in `lib/payment.ts`
